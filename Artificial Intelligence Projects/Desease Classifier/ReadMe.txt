disease.exc: This is a disease dataset where each row is information for each patient and the first 8 columns are 8 different characteristics where the last column indicating whether the patient has 'the disease' or not.

utilities.py: contains all the functions called in algorithms.py and script_classify.py.

algorithm.py: the naive-bayes and logistic regression algorithms are developed there.

script.py: main file. The classifying algorithms are carried out on the given disease dataset.
